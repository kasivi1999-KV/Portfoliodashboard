⇅ Data Sync & Backup
All data saves to this browser's localStorage. To use across devices, export JSON → save to Google Drive → import on other device.
Cross-device usage guide
1
Save the HTML file to Google Drive
Upload portfolio-command-center-v3.html to your Google Drive. This is your app file — it never changes unless you update it.
2
Export data after each session
Click "Export All Data" → saves a .json file to your Downloads. Drag this to a folder in Google Drive (e.g., "Portfolio Data → latest.json"). Takes 5 seconds.
3
On another device (laptop/tablet)
Open Google Drive → download the HTML file → open in browser → click Import → select the latest.json from Drive. Your entire portfolio loads instantly.
4
On Mobile (iPhone/Android)
Google Drive app → open HTML → tap ⋮ → "Open in Browser". Then import JSON. OR use Chrome → open HTML → works the same way. Tip: bookmark it in Chrome on mobile.
5
Alternative: GitHub Pages (advanced)
Upload HTML to a GitHub repo → enable GitHub Pages → get a permanent URL (free). Works on any device, any browser, without downloading anything. Share URL only with yourself.
